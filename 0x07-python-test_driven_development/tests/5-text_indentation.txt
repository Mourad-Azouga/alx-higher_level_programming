
================================
How to Use 5-text_indentation.py
================================

This modue is for  ``text_indentation(text)``.

Usage
=====

Text is printed with two new lines after ``.``, ``?``, and ``:``.

::

    >>> text_indentation = __import__('5-text_indentation').text_indentation
    >>> text_indentation("Hello?")
    Hello?
    <BLANKLINE>

No spaces should printed at the beginning of a line.

::

    >>> text_indentation("   Hi there.")
    Hi there.
    <BLANKLINE>

::

    >>> text_indentation("          ")

Similarly, no spaces are printed at the end of each printed line.

::

    >>> text_indentation("Testy test.   ")
    Testy test.
    <BLANKLINE>

::

    >>> text_indentation("    Woah now.    This is messy.   ")
    Woah now.
    <BLANKLINE>
    This is messy.
    <BLANKLINE>


New lines within a string are printed as normal.

::

    >>> text_indentation("Mixed:\nMixy cases.")
    Mixed:
    <BLANKLINE>
    <BLANKLINE>
    Mixy cases.
    <BLANKLINE>

::

    >>> text_indentation("\n\n Manually printing lines.")
    <BLANKLINE>
    <BLANKLINE>
    <BLANKLINE>
    Manually printing lines.
    <BLANKLINE>

::
    >>> text_indentation("A sneaky \n new line.")
    A sneaky 
    new line.
    <BLANKLINE>

Invalid Text
============

The paramter ``text`` must be a string. Otherwise, a TypeError is raised.

::

    >>> text_indentation(7)
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation({"one": 1, "two": 2})
    Traceback (most recent call last):
    TypeError: text must be a string

::

    >>> text_indentation(None)
    Traceback (most recent call last):
    TypeError: text must be a string
